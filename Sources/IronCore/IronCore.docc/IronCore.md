# ``IronCore``

Foundation module providing theming, tokens, and accessibility utilities.

## Overview

IronCore is the foundation of IronUI, providing the theming system, design tokens, and core utilities used by all other modules. It defines the visual language and ensures consistency across the entire component library.

## Topics

### Theming

Configure the visual appearance of IronUI components.

- ``IronTheme``
- ``IronDefaultTheme``

### Color Tokens

25 semantic colors with automatic light/dark mode adaptation.

- ``IronColorTokens``
- ``IronDefaultColorTokens``

### Typography Tokens

15 font styles following a consistent typographic scale.

- ``IronTypographyTokens``
- ``IronDefaultTypographyTokens``

### Spacing Tokens

10 spacing values following an 8pt grid system.

- ``IronSpacingTokens``
- ``IronDefaultSpacingTokens``

### Radius Tokens

7 corner radius values from sharp to fully rounded.

- ``IronRadiusTokens``
- ``IronDefaultRadiusTokens``

### Shadow Tokens

5 elevation levels with multi-layer shadows.

- ``IronShadowTokens``
- ``IronDefaultShadowTokens``

### Animation Tokens

Duration and spring animation presets.

- ``IronAnimationTokens``
- ``IronDefaultAnimationTokens``

### Logging

Structured logging for debugging.

- ``IronLogger``
