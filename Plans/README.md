# Plans Directory

This directory is the source of truth for planning work in IronUI.

## Structure

- `Plans/Roadmaps/` - High-level, multi-release planning.
- `Plans/Workstreams/` - Focused plans for a specific initiative.

## Conventions

- Use concise filenames with hyphens (e.g., `docc-unification.md`).
- Include: Goals, Non-Goals, Milestones, Risks, and Next Actions.
- Migrate legacy planning docs into this directory.
